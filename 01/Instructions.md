There are two ways to see your "Movie App" or "Chipsi Mayai" recipe:

Locate the file in your computer's folder.

Double-click it. It will open in your default browser (Chrome, Safari, or Edge).

Method B: The Pro Way (VS Code Extension)
In VS Code, click the Extensions icon on the left sidebar (or Ctrl+Shift+X).

Search for "Live Server" by Ritwick Dey.

Click Install.

Once installed, look at the bottom-right corner of VS Code and click "Go Live".

This will launch a local server and open your browser automatically. Any changes you save in the code will update the page instantly!
